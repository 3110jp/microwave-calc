# 電子レンジ調理時間換算器

冷凍食品に記載されたワット数・調理時間を入力すると、自分の電子レンジに合わせた調理時間を算出するWebアプリです。

🌐 [公開URL](https://3110jp.github.io/microwave-calc/)

## 使い方

1. **食品に記載されたワット数**を選択（500W・600W・700W・その他）
2. **食品に記載された調理時間**の分・秒を入力
3. **あなたの電子レンジのワット数**を入力（デフォルト 800W、入力値は記憶される）
4. 換算後の調理時間が即座に表示される

## 換算の原理

エネルギーは「電力 × 時間」で一定なため、

```
T₂ = W₁ × T₁ ÷ W₂
```

で換算できます。結果は安全側に**切り上げ（秒単位）**で表示されます。

| 変数 | 意味 |
|------|------|
| W₁ | 食品に記載されたワット数 |
| T₁ | 食品に記載された調理時間 |
| W₂ | あなたの電子レンジのワット数 |
| T₂ | 換算後の調理時間 |

## 技術構成

単一の `index.html`（HTML + CSS + JS一体）で構成されます。外部依存なし、ビルドステップ不要です。
